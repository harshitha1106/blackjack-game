# Blackjack Game

A simple Blackjack game built with HTML, CSS, and JavaScript. In this game, the player draws cards and tries to reach a sum of 21 without going over.

## Features

- Start a new game
- Draw a new card
- Random card generation
- Score calculation
- Win/lose game messages
- Simple and clean user interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Play

1. Click **START GAME** to begin.
2. Two random cards will be drawn.
3. If your total is less than 21, you can click **NEW CARD** to draw another card.
4. If your total becomes exactly 21, you win with Blackjack.
5. If your total goes above 21, you lose the round.
